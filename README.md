# PG Finder Database Management System

## Project Overview

PG Finder Database Management System is a Java and MySQL-based application designed to manage Paying Guest (PG) accommodation information efficiently. The system helps store, update, search, and manage PG details, including room availability, rent, location, and tenant information.

## Features

* Add new PG records
* Update existing PG details
* Delete PG records
* Search PGs by location or other criteria
* Manage tenant information
* Store and retrieve data using MySQL database
* User-friendly menu-driven interface

## Technologies Used

* Java
* JDBC
* MySQL
* SQL

## Database Operations

The project implements CRUD operations:

* Create: Add new PG records
* Read: View and search PG information
* Update: Modify existing records
* Delete: Remove records from the database
